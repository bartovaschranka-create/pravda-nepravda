# Citace v čase

Statické MVP webové aplikace pro GitHub Pages.

## Cíl

Aplikace zobrazuje veřejné výroky, rozhovory, související články a události v časové ose. Nehodnotí osoby a neurčuje pravdivost výroků.

## Aktuální MVP

- pole Osoba,
- pole Klíčová slova,
- cílené odkazy na veřejné vyhledávací zdroje,
- ukázková časová osa,
- neutrální shrnutí souvislostí.

GitHub Pages nemůže bez backendu spolehlivě procházet webové vyhledávače ani obcházet CORS. První verze proto funguje jako transparentní statické rozhraní a základ pro další napojení na vyhledávací API nebo ručně kurátorovaná data.

## Spuštění

Otevřete `index.html` v prohlížeči nebo nasaďte repozitář na GitHub Pages.
